# Terms of Reference 1: Bash-скрипт для копирования файлов

Этот скрипт позволяет скопировать все файлы из входной директории (и её поддиректорий) в выходную директорию без сохранения иерархии.

## Использование и тестировка

1. **Клонирование репозитория:**

    ```bash
    git clone https://github.com/alex-bul/hse_tp_task1.git
    ```

2. **Запуск скрипта:**

    Перейдите в каталог с клонированным репозиторием и выполните:

    ```bash
    bash copy.sh <входная_директория> <выходная_директория>
    ```

3. **Пример входной дирректории для тестирования:**

   ```bash
    bash copy.sh /path/to/input_directory /path/to/output_directory
    ```
